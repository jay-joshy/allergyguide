# Privacy Policy

## 1. Data Collection

This website is a static site designed for educational purposes. We do not use cookies, trackers, or analytics software to monitor your behaviour. We do not knowingly collect, store, or sell any personal information or Patient Health Information.

This site is hosted on Netlify. While we do not collect data, Netlify may collect standard server access logs (such as IP addresses) for security, performance, and reliability monitoring. You can view Netlify's privacy policy [here](https://www.netlify.com/privacy/).

## 3. External Links

This website contains links to third-party websites. We are not responsible for the privacy practices or content of those external sites.

The source code for this website is open-source and available for review on GitHub. You are welcome to inspect how this site is built.
